## PDCA/PDF
### PDCA
- plan 计划
- do 做
- check 检查
- action 改善
```mermaid
graph LR;  
　　plan -->do;    
　　do -->check;  
　　check-->action;  
　　action-->plan;  
```


### PDF
- preview 沙盘
- do 做
- fupan 复盘
```mermaid
graph LR;  
    preview -->do;    
　　do -->fupan;  
　　fupan-->preview;  
```
