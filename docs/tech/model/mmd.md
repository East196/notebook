### 3.[b碗](https://bowlroll.net/)下载动作

### 4.模型和场景下载
- 模型移动
点击全部选择，再点移动（有事移动不能点，那就先点模型的方形骨骼，再点全部选择，再点击移动），人物脚下出现坐标轴。移动后需要注册，否则不会保存。
- 借物表写法
model：model x by author
motion：motion x by author
song：song x by author
